# TDOS Engineering Governance Model

## Status

BASELINE SUMMARY

## Flow

```text
Engineering Evidence
  ↓
Candidate Recommendation
  ↓
Runtime Review
  ↓
Architecture Review, if required
  ↓
Authorized Decision
  ↓
Controlled Integration
```

## Constraints

- Simulator output is not an authorized decision.
- Architecture changes require Architecture Governance.
- Governance Compatibility is separate from Runtime Quality Criteria.
