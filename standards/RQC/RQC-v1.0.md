# Runtime Quality Criteria (RQC)

## Status

FROZEN

| ID | Criterion | Objective |
|---|---|---|
| RQC-01 | Determinism | Equivalent inputs produce equivalent runtime semantics |
| RQC-02 | Completeness | Required behavioural content is present |
| RQC-03 | Ambiguity | Independent implementations reach the same interpretation |
| RQC-04 | Traceability | Behaviour remains traceable to governing artifacts |
| RQC-05 | Implementability | No undocumented architectural assumptions are required |
| RQC-06 | Verifiability | Behaviour is objectively verifiable |

Governance Compatibility is a separate check and is not an RQC.
